# ZEPTO INVENTORY ANALYSIS
## Project Overview
The Zepto Inventory Analysis Dashboard is an interactive Power BI dashboard designed to analyze inventory performance, pricing, discounts, stock availability, and product distribution across different categories. The dashboard provides actionable insights to help inventory managers optimize stock levels, monitor pricing strategies, and improve overall inventory efficiency.

## Objectives
Analyze inventory distribution across different price ranges.
Monitor stock availability and out-of-stock products.
Understand discount patterns across products.
Compare MRP with discounted selling prices.
Analyze product weight distribution.
Support data-driven inventory management decisions.

## Key Performance Indicators (KPIs)
📦 Total SKUs
💰 Total Inventory Value
💵 Average MRP
🏷 Average Discount %
✅ Total In-Stock Products
❌ Total Out-of-Stock Products
📉 Out-of-Stock Rate (%)

## Dashboard Features
1. Price Range Distribution

Shows the number of products available in different price buckets.

2. Products by Price Category

Compares product count across various price ranges.

3. Weight Distribution

Analyzes products based on their weight categories.

4. MRP vs Discounted Price

Scatter plot comparing original price with discounted selling price while highlighting stock status.

5. Stock Availability

Donut chart displaying In-Stock vs Out-of-Stock products.

6. Discount Distribution

Visualizes products across different discount percentage ranges.

7. Interactive Category Filter

Allows users to filter the entire dashboard by product category.
